## 🚀 How to Use This Repository

1. **Clone the repository:**
   ```bash
   git clone https://github.com/UttkarshRithe/firstbitsolutions.git
   cd firstbitsolutions
   ```

2. **Navigate to the required module:**
   ```bash
   cd "c language/Assignments/assignment0-ArithmeticOperator"
   # or
   cd "Core Java/LAB"
   # or
   cd "Data Structure/Lab/Stack"
   ```

3. **Compile and run programs:**
   - **C Programs:**
     ```bash
     gcc Q1.c -o Q1
     ./Q1
     ```
   
   - **Java Programs:**
     ```bash
     javac ClassName.java
     java ClassName
     ```
   
   - **Python Programs:**
     ```bash
     python filename.py
     ```
